# ファイル名を PascalCase クラス名に合わせるリネーム

クラス・主要構造体と一対一対応するファイルについて、ファイル名をクラス名そのまま (PascalCase) にリネームする。
関数モジュールや複数構造体の集合ファイルは対象外。
